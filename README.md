# StudyMate AI Tutor

StudyMate is an AI-powered study assistant that creates
explanations, quizzes, flashcards, summaries, and study plans.

## Technologies

- Python
- Object-oriented programming
- Pandas
- Gemini Interactions API
- Streamlit
- GitHub

## Features

- Multi-turn AI conversations
- Personalized tutoring instructions
- Study-history tracking
- Pandas-based statistics
- CSV downloads
- Interactive Streamlit interface

## OOP Design

- `StudyMateBot` manages Gemini interactions.
- `StudyHistory` manages Pandas data and analytics.
- Composition connects the chatbot to its study-history object.
